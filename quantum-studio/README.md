# Quantum Programming Studio

**Quantum Programming Studio** is an interactive web-based tool for designing and simulating quantum circuits. Built with React and TypeScript, it offers a drag-and-drop interface to construct quantum algorithms visually.

## ğŸ“¦ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

*   [Node.js](https://nodejs.org/) (v14 or higher)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/VarunRathore137/Quantum_Programming_Studio.git
    cd quantum-studio
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the App

Start the development server:

```bash
npm start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
